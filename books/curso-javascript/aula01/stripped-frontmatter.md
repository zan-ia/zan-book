# JavaScript — Do Zero ao Profissional Aula 01 — Questoes de Aprendizagem

## Como Usar Este Arquivo

Este arquivo contem **questoes de aprendizagem** para a Aula 01.

Diferente dos exercicios que voce fez durante a aula, estas questoes funcionam como um **checkpoint de dominio** — voce deve ser capaz de responde-las SEM consultar o conteudo principal. Se precisar consultar, volte e revise a secao indicada.

**Instrucoes:**
1. Leia cada questao com atencao
2. Siga os passos de execucao
3. Preencha o template de entrega em um arquivo separado (ou no seu caderno)
4. So avance para a Aula 02 quando conseguir completar todas as questoes sem reler a aula

> **Dica:** Crie uma pasta chamada `meus-estudos/aula01/` para guardar suas respostas. Voce vai consulta-las nas aulas seguintes.

---

## Questao 1: Escrevendo uma "Receita-Programa"

**Conceito-chave:** Aula 01, Secao 1 — O Que E Um Programa?

**Objetivo:** Demonstrar que voce entendeu o que e um programa usando a analogia da receita e identificando o fluxo Entrada, Processamento e Saida.

**Passos de Execucao:**

1. Pense em uma atividade cotidiana que voce faz seguindo passos (fazer um sanduiche, preparar um suco, organizar sua mochila, carregar o celular)
2. Escreva essa atividade como uma "receita-programa", listando os passos em ordem
3. Identifique qual parte e a "entrada", qual e o "processamento" e qual e a "saida"

**Entrega:** crie `entregas-aula01/01-receita-programa.md`:

~~~~
# Questao 1 — Minha Receita-Programa

## Atividade escolhida
[escreva aqui]

## Passos
1. [passo 1]
2. [passo 2]
3. [passo 3]
4. [passo 4]

## Fluxo Entrada, Processamento e Saida
- **Entrada:** [o que voce precisa para comecar]
- **Processamento:** [os passos que voce executa]
- **Saida:** [o resultado final]

## Conclusao
Em 2-3 frases: como esta atividade se parece com um programa de computador?

## Questao 7: Lendo Mensagens de Erro no Console

**Passos de Execucao:**

1. Abra o console do navegador (F12)
2. Digite cada comando abaixo com o erro INTENCIONAL e observe a mensagem vermelha que aparece:

   - `console.log("Ola);` — aspas abertas mas nao fechadas
